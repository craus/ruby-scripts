Это пример файла, на котором программа FileRecoder может обучаться. Она будет использовать статистический анализ файлов в этой папке, чтобы познать закономерности текстовый файлов на естественном языке. Получившуюся статистика программа будет использовать для придумывания текстов, выглядящих естественно и для эффективного сжатия естественных текстов.
Второй абзац добавлен в этот файл исключительно с целью показать программе, что бывают переводы строки.
