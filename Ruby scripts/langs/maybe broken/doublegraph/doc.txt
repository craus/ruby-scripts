set x1 x2 y1 y2 z1 z2 w1 w2 
a[a[x1,x2],a[y1,y2]] = a[a[z1,z2],a[w1,w2]]

a[`,x] = x forall x

new x1 x2 y1 y2
a[a[x1,x2],a[y1,y2]] = <new cell>

Если первый токен строки - use, остальные - имена файлов для прочтения
Иначе, если в строке есть токен "=", первые 2 токена - объекты x и y, 4-й токен - объект z, и a[x,y] = z
Иначе, это строка кода.
Если первый токен заканчивается на ":", это название строки кода, иначе название строки кода - очередное натуральное число
Поле next предыдущей строки кода становится равным этому названию.
Второй токен - это команда строки кода.
Остальные токены - это аргументы команды, они называются arg, arg2, arg3, ..., arg8

Возможные команды:
  write(x) 
    пишет символ x на консоль
  read() 
    читает с консоли символ, кладёт в result value
  set x1 x2 y1 y2 z1 z2 w1 w2 
    a[a[x1,x2],a[y1,y2]] = a[a[z1,z2],a[w1,w2]]
  new x1 x2 y1 y2
    a[a[x1,x2],a[y1,y2]] = b = <new cell>
    a[`,b] = b

Предустановленные объекты:
  `
    a[`,x] = x forall x
  команды
  символы 'c', где с - любой символ, кроме пробела и перевода строки
  space - пробел
  endl - перевод строки
  
  next
  arg
  arg2 ... arg8
  command
  result
  value
  nil - ему всё равно
